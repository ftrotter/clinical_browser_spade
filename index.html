Chrome Browser history location hints

  • On Windows

    • On Windows XP: C:\Documents and Settings\USERNAME\Local Settings\Application Data\Google\Chrome\User Data\Default\History
    • On Windows VistA and later: C:\Users\USERNAME\AppData\Local\Google\Chrome\User Data\Default\History

  • On OSX

    You are looking for a file called "History". On different versions of OSX, it lives in different places. Usually under the Library directory off of the home directory somewhere. Remember that ~/Library can be a hidden directory! Here are video instruction to find the file!

    • ~/Library/Caches/Google/Chrome/Default/History
    • ~/Library/Application Support/Google/Chrome/Default/History
    • ~/Library/Caches/RockMelt/Default/History

  • On Linux

    • You use this operating system because you can figure it out on your own (its under one of the hidden '.' directories under your home directory)

Firefox Browser history location hints

  • On Windows

    • On Windows XP: C:\Documents and Settings\USERNAME\Local Settings\Application Data\Firefox\Profiles\RANDOMTEXT\places.sqlite
    • On Windows VistA and later: C:\Users\USERNAME\AppData\Mozilla\Firefox\Profiles\RANDOMTEXT\places.sqlite

  • On OSX

    • ~/Library/Application Support/Firefox/Profiles/RANDOMTEXT/places.sqlite

  • On Linux

    • You use this operating system because you can figure it out on your own (its under one of the hidden '.' directories under your home directory)
